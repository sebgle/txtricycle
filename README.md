
# Texas Triathlon & Cycling Club website

A responsive web app built with Next.js and Tailwind CSS to showcase the Texas Triathlon and Cycling Clubâ€™s events and community.

## Features

- Responsive design for both desktop and mobile
- Event listings with details on upcoming triathlons
- Information about the Texas Triathlon Club
- Easy access to membership and registration info

## Technologies Used

- Next.js
- Tailwind CSS 
- TypeScript

## Installation

1. Clone the repo:
   ```
   git clone https://github.com/sebgle/txtricycle.git
   ```
2. Go into the project folder:
   ```
   cd txtricycle
   ```
3. Install dependencies:
   ```
   npm install
   ```

## Development

To run the dev server:
```
npm run dev
```
Then open [http://localhost:3000](http://localhost:3000) in your browser.

## Testing

To run tests:
```
npm run test
```
